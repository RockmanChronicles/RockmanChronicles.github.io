# Rockman Chronicles (RC) 洛克人馬拉松活動頁面

![Rockman Chronicles Logo](assets/img/TitleRC_Black.png)

這是一個由熱愛《洛克人》系列的實況主們共同發起的社群活動。每年七月，我們聚集在一起，透過每日接力實況的方式，帶領觀眾重溫童年的感動。

## 🚀 活動特色
- **七月限定**：每日晚上準時開台，帶來不同系列的洛克人挑戰。
- **多樣化內容**：包含技術台、Speedrun（RTA）、Vtuber 參與以及各種趣味挑戰。
- **跨平台串聯**：集結 Twitch / Youtube 多位知名實況主共同參與。
